- [ ] **要件定義・計画**:
    - [ ] 詳細な仕様を確認する
    - [ ] 実装プランを作成する
- [ ] **実装**:
    - [ ] クラス/機能Aの実装
    - [ ] クラス/機能Bの修正
- [ ] **テスト・検証**:
    - [ ] 動作確認を行う
    - [ ] バグ修正
